# luas-bangun-datar
Simple Java Program untuk mencari Luas bangun datar

## Usage
Pilih Bangun Datar :
1. Persegi
2. Segitiga
3. Lingkaran

Ketik (1/2/3), Enter.
Kemudian
jika Persegi, masukkan value Sisi.
jika Segitiga, masukkan value Alas & Tinggi.
jika Lingkaran, masukkan value Jari-jari.

Enter, hasil akan ditampilkan
